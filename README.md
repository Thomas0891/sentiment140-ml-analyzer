# 📖 Overview

This project performs **Twitter Sentiment Analysis** using the famous **Sentiment140 Dataset** containing **1.6 Million Tweets**.

The project follows an end-to-end Machine Learning workflow:

- Data Loading
- Data Cleaning
- NLP Preprocessing
- Feature Engineering
- Exploratory Data Analysis
- Visualization
- TF-IDF Vectorization
- Machine Learning Model Training
- Model Evaluation
- Streamlit Deployment
- Live Sentiment Prediction

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Sentiment140-Analyzer.git

cd Sentiment140-Analyzer
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Download Dataset

Download the **Sentiment140 Dataset**

Place it inside:

```
data/
```

Expected filename:

```
training.1600000.processed.noemoticon.csv
```

---

# ▶ Run Complete Pipeline

```bash
python main.py
```

---

# 🌐 Run Streamlit Application

```bash
streamlit run app.py
```
